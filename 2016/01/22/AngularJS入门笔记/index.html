4.AngularJS 表达式 与 JavaScript 表达式

相同点
  • 类似于 JavaScript 表达式，AngularJS 表达式可以包含字母，操作符，变量。
    #####不同点
  • 与 JavaScript 表达式不同，AngularJS 表达式可以写在 HTML 中。
  • 与 JavaScript 表达式不同，AngularJS 表达式不支持条件判断，循环及异常
  • 与 JavaScript 表达式不同，AngularJS 表达式支持过滤器。

2.AngularJS指令

1.一些初级的AngularJS指令

1.ng-app指令

该指令定义了AngularJS-APP的根元素.在ng-app指令作用域以外的部分不会受到AngularJS的影响.

2.ng-init 指令

该指令定义了AngularJS-APP的变量的初始值。

3.ng-model指令

ng-model指令绑定HTML元素到APP数据。不仅如此，其实它还有更多的用法。

4.ng-repeat指令

ng-repeat 指令对于集合中（数组中）的每个项会 克隆一次 HTML 元素。

5.ng-bind指令

ng-bind指令可以将APP数据中的变量绑定到HTML元素的内容。
